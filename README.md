
## Setting

### 1. Tailwindcss

- Yon Need to [Install Tailwindcss](https://tailwindcss.com/docs/installation).

### 2. Global css

- Import

```typescript

import '@actionpower/daglo-ui/dist/styles/calendarStyle.css';
```
